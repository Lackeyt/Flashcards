JS & JQuery Flashcards

JavaScript

An interpreted programming language for adding interactivity to webpages.

Operators

+ - / * %

Variables

A way to store data values. (var, const, let)

Variable Naming Conventions

Use const if a variable will remain unchanged, or let if the value will change. Use camelCase.

Functions

A statement to perform a task.

Methods

Can be called on a object to alter the value.

Arguments

A value passed into a function when calling the function.

Parameters

A value used when defining a function; a placeholder.

Return

A statement used within a function to return a value and stop execution of the function.

Chaining Methods

Placing multiple methods on an element with a single statement.

Strings

Numbers or letters contained within quotes.

Booleans

True or false.
